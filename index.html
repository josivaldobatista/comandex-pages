Política de Privacidade - Comandex

O Comandex é um sistema de gestão para restaurantes.

Coletamos informações necessárias para autenticação e funcionamento da plataforma.

Os dados cadastrados podem incluir informações da empresa, produtos, pedidos, mesas e usuários autorizados.

Os dados são utilizados exclusivamente para operação do sistema.

Não comercializamos dados pessoais com terceiros.
